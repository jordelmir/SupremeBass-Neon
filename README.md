## 🏗 Build & Usage

To compile the project from source:

1. Clone the repository.
2. Open in Android Studio (Jellyfish or newer recommended).
3. Ensure you have JDK 17+ configured.
4. Run `./gradlew assembleDebug` to generate your own build.

> [!CAUTION]
> **HARDWARE WARNING**: Using boost levels above 200% (High Risk/Nuclear Stage) can cause permanent damage to mobile speakers. Use with extreme professional caution.
